Structural Elements

HTML5 introduced new structure element to arrange certain areasnbased on their use:

Tag Name Description
<div> Div This element creates "dividers" within a specific areas of the webpage. It places empty spaces above and below the content within.
<header> Header element This is generally first structur element within the document. It generally contains heading elements and it's text.
<nav> Nav Element THis area contains navigation links for locating information within the website. This element can also contain links to external websites.
<main> Main Element It contains the main content of the webpage and should only be used once per document.
<footer> Footer Element The footer section contains general links and Copyright/Trademark information.

Other helpful Elements

Tag Name Description
<figure> Figure Element The <figure> element is used to encapsulate media such as an image, diagram.
<figcaption> Figcaption Element The <figcaption >element is used to describe the media encapsulated within the <figure> element.
